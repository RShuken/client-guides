windows-approval-gates -- safety for destructive actions Deployed

Five gated action categories. Before Clawd does anything in these categories, it pauses, shows you the command + rollback plan, and waits for approval.

  • Outbound messages (emails, posts, DMs to unknown recipients)
  • File / folder deletion (prefers Recycle Bin over permanent delete)
  • Registry writes (shows old and new value before changing)
  • Scheduled Task creation / edit / delete
  • Execution of downloaded binaries (shows URL + signing info first)

Also includes financial-data lockdown (keeps $ amounts in DMs only, never in groups) and prompt-injection defenses.

kb-nomic -- private semantic knowledge base Deployed (custom-built)

Full local RAG pipeline, written from scratch in PowerShell. Zero external dependencies beyond what's already on your machine (Ollama for embeddings, JSONL for storage).

  • kb-ingest.ps1 -- fetches URL, strips HTML, chunks 500 words with 50-word overlap, embeds each chunk via nomic, appends to store.jsonl
  • kb-search.ps1 -- embeds a query, computes cosine similarity against every stored chunk, returns top-K
  • kb-ask.ps1 -- full RAG: search + pass retrieved chunks to local Gemma as context, generate cited answer
  • Storage: C:\Users\paul\.openclaw\workspace\kb\store.jsonl
  • Currently holds 8 chunks from the Gemma 4 model card (test ingest)

Verified end-to-end: query "how many parameters does E4B have" returned the correct chunk with score 0.5977, Gemma generated a cited answer at 22.8 tok/s and honestly noted the context didn't cover the context window question (refused to make up a number).

What's not done (and why)

Personalize Clawd's identity Pending

Clawd is currently using the stock "lobster energy" identity from the template. Takes a ~10-minute intake conversation to customize -- name, emoji, creature metaphor, tone, humor style, no-go topics. Can do this any time via @PauloradoBot directly.

Google Workspace (Gmail / Calendar / Drive) Deferred

The OpenClaw Google Workspace integration uses a Linux/macOS-only CLI tool called gog. Needs a Windows-specific adaptation. Not a blocker today -- if/when you need Clawd to see your email or calendar, we'll build the Windows path.

PDF ingestion into KB Deferred

Current kb-ingest.ps1 handles URLs (HTML). PDFs need a text extraction step via pypdf or pdftotext. Easy add once you have specific PDFs you want searchable.

Secrets migration Deferred

Telegram bot token is currently stored in plaintext in openclaw.json. This is fine for a local-only file on your user account, but the cleaner setup uses OpenClaw's dedicated secrets subsystem. Defer-worthy -- no functional impact.

openclaw capability model run --model ollama/... CLI hang Known issue, non-blocking

This specific CLI command hangs at the 120-second timeout when pointed at the local Ollama model. Likely related to Gemma 4's default thinking mode and OpenClaw's streaming adapter. Doesn't affect production: the Telegram agent loop uses a different internal code path that works correctly. The KB's kb-ask.ps1 bypasses this by calling Ollama's HTTP API directly.
